# Online Job Portal

An online job portal is a simple and user-friendly platform where job seekers can find jobs based on their skills, experience, and roles, while recruiters can post job openings and hire suitable candidates easily.

## Features
- Job search based on skills and experience
- Recruiter job posting
- Clean and responsive UI
- Built with modern web technologies

## Tech Stack
- React (TypeScript / TSX)
- Vite
- Tailwind CSS
- shadcn-ui

## How to Run Locally
```bash
npm install
npm run dev
